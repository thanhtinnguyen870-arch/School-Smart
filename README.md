# SMART SCHOOL AI - Student Face Attendance Management System

Full Stack MERN MVP for school management, face attendance demo, grades, assignments, online tests, reports, and notifications.

## Run

Backend:

```bash
cd backend
npm install
npm run seed
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Accounts:

- Admin: `admin@gmail.com` / `123456`
- Student: `student@gmail.com` / `123456`

MongoDB default: `mongodb://127.0.0.1:27017/smart_school_ai`
